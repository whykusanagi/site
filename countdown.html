Countdown Widget

This page renders the site countdown component using JSON presets from static/data/countdown/ plus optional URL overrides.

Quick Start

  • ?event=kirara loads kirara.json.
  • ?event=bastard-hero loads bastard-hero.json.
  • ?event=default loads the generic template preset.

Common URL Parameters

Use event, theme, character, overlay, shape, date, and title. URL parameters override values from the JSON file.

Quick FAQ

Q: Where are preset files stored?
A: In static/data/countdown/.

Q: Can I test alternate shapes quickly?
A: Yes. Use presets like test-circle, test-hexagon, test-star, and test-heart.

Q: What if a URL value is invalid?
A: Invalid preset values fall back to defaults.
